
# PATIENT MANAGEMENT SYSTEM
#### By **evans mwenda**
## Description
This is a web application that allows doctors to manage their patients' data as well as allows the patients to manage appointments with their doctors.
## Setup/Installation Requirements
* install ven in your machine
* git clone on the github account
* you can check the requirements.txt file for more required installation.
* then navigate to terminal and activate the virtual environment
* ./manager.py runserver

## As a user i should be able to:
* Sign in to the application to start using.
* Crete a profile as a doctor or patient.
* View profiles of their patient if its a doctor.
* Patients can view profiles of different doctors
## Known Bugs
the application is under development phase especially so expect some changes
